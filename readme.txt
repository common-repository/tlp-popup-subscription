TLP Popup Subscription is the smart, responsive, customizable and beautifully coded pop-up plugin.

== Description ==

TLP Pop Up Subscription is an advanced wordpress pop-up plugin. Fancybox popup system is added with this system with different customizable settings. Any kind of HTML code can be displayed by this plugin.

= Features =
* Fully Responsive
* Any kinds of Html code can be used
* Advanched Settings
* Custom POPUP display time
* Session validation time
* width and height responsive settings

== Installation ==

= Install and Activate =

1. Unzip the downloaded 'tpl-popup-subscription' zip file
2. Upload the 'tpl-popup-subscription' folder and its contents into the 'wp-content/plugins/' directory of your WordPress installation
3. Define yourpopup settings

= Requirements =
* **WordPress version:** >= 4
* **PHP version:** >= 5.2.4


== Frequently Asked Questions ==

Q : How can I use this plugin?
A : Go to the Settings -> TLP popup
